### For Cleanup Experts
> "DataForge duplicate detection superpowers: Find and remove 2.3 TB of duplicates in 45 seconds. Reclaim your storage."
```bash
fm dupes ~/10tb-archive --sort size --execute
```

### For IT Auditors
> "Integrity superpowers: Create SHA-256 baselines, verify tampering in seconds, generate audit-ready reports."
```bash
fm integrity create ~/critical-data snapshot.json
fm integrity check ~/critical-data snapshot.json
```

### For Forensics Analysts
> "File carving superpowers: Recover 847 JPEGs from disk image using file signatures. Generate forensic report with hashes."
```bash
fm recover --carve /dev/sdb1 --out ~/Recovered --types jpg,png,pdf
```

### For Developers
> "Automation superpowers: Build multi-step workflows, batch process thousands of files, integrate via JSON API."
```bash
fm search ~/media --ext jpg,png --sort size --format jsonl | jq '.path'
```

### For Power Users
> "System cleanup superpowers: One command removed 47 GB of cache, duplicates, and temp files. Computer runs faster."

## FAQ: Superpowers Branding

**Q: Is "steroids and superpowers" too playful?**  
A: No. It's bold and confident, which signals actual power. Enterprise users respect this more than generic "professional" language. It's memorable without being unprofessional.

**Q: Will this alienate security/compliance professionals?**  
A: No. In formal compliance/audit docs, use "enterprise-grade" and "production-tested" instead. The tagline is for marketing and product positioning. The substance (224 tests, thread-safe, audit trails) backs up the confidence.

**Q: Should we use this in commit messages?**  
A: Optionally. Use normal commits, but can say "Add forensic carving superpowers for JPEG recovery" instead of just "Add JPEG carving". It's about tone, not required everywhere.

**Q: Can we change it later?**  
A: Yes, but consistency builds brand recognition. Stick with "steroids and superpowers" for at least 6-12 months before reconsidering.
